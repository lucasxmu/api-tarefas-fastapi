# API de Tarefas - FastAPI + MongoDB + Docker

## 📌 Descrição
Este projeto consiste em uma API backend desenvolvida com FastAPI, utilizando MongoDB como banco de dados e Docker para containerização.

A aplicação permite realizar operações CRUD (Create, Read, Update, Delete) de tarefas.

## 🧱 Arquitetura
O projeto segue uma arquitetura modular dividida em:

- routers → rotas da API
- services → lógica da aplicação
- repositories → acesso ao banco de dados
- schemas → validação de dados

## 🛠️ Tecnologias utilizadas
- FastAPI
- MongoDB
- PyMongo
- Docker
